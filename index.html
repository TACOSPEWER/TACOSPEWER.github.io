Inputs

  • For room numbers, capitalize mod letter with no spaces (Ex - “A105”)
  • For the Theater, type “A148” or “theater”
  • For the Robotics lab, type “A155”, “A157”, or “rob”
  • For the Gymnasium, type “B114” or “gym”
  • For the Tefft Library, type “B137” or “tefft”
  • For the B Lobby, type “lobby”
  • For the Directors’ Office, type “dir”
  • For the East Cafeteria, type “caf”
  • For the Copying Room, type “copy”
  • For the Junior/Senior Area, type “jrsr”
  • For the A Mod Bathrooms, type “abath”
  • Otherwise, refer to the key for room numbers.
